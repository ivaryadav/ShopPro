# ShopERP Pro v8 — Electron Desktop App

## How to Run

### Prerequisites
Install Node.js from https://nodejs.org (LTS version)

### Steps
1. Unzip this folder
2. Open terminal / command prompt in this folder
3. Run:
   npm install
4. Then run:
   npm start

ShopERP Pro will open as a standalone desktop window!

## Build Installer (optional)

To build a .exe installer for Windows:
   npm install
   npm run build-win

The installer will be in the dist/ folder.

To build for Mac:
   npm run build-mac
